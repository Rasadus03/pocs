PURPOSE :  provide isolated labs on various JBoss EAP 6 / JEE technologies

PRE-REQUISITES :

  1)  ant 1.8.2 with latest ant-contrib.jar in $ANT_HOME/lib/
  2)  $JAVA_HOME environment variable pointing to jdk 1.6 (openjdk recommended)
  3)  $JBOSS_HOME environment variable pointing to latest JBoss EAP 6

PROCEDURE :

  1)  change directory into the lab of interest
  2)  read the lab specific README.txt
